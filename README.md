# DescubreSV-AppWeb

## Descripción del Proyecto

DescubreSV es una solución digital diseñada para optimizar la planificación de viajes en El Salvador.
La aplicación permite calcular presupuestos personalizados y generar itinerarios basados en destinos emblemáticos, 
integrando seguridad robusta y una interfaz dinámica para mejorar la experiencia del turista nacional e internacional.
